# act-runner-images

This repository contains customized gitea runner images based on the upstream images at
https://gitea.com/gitea/runner-images

## images

* ubuntu-24.04 - ghcr.io/eoscloud/act-runner-images:ubuntu-24.04

## additional packages

* podman
* buildah
* containers-storage
* iptables
* qemu-user-static
* skopeo
